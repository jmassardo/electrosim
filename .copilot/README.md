# ElectroSim Custom Instructions

## Project Overview
This directory contains custom instructions and prompts for AI agents working on the ElectroSim project.

## Files

### `instructions.md`
Comprehensive development guidelines and standards for the ElectroSim project. Contains:
- Agent identity and expertise requirements
- Code quality standards and testing requirements
- Development workflow and autonomous process
- Project architecture knowledge
- Component development guidelines
- Performance and security requirements

### `prompt.md`
Focused prompt template for engaging AI agents in autonomous development work. Designed to:
- Establish expert software and electronics engineering identity
- Set expectations for independent iteration through work items
- Define completion criteria and workflow standards
- Minimize need for human interaction during development cycles

## Usage

### For Development Sessions
When starting a development session, reference both files to ensure the agent:
1. Understands the full technical requirements (`instructions.md`)
2. Adopts the proper autonomous working approach (`prompt.md`)

### For Task Assignment
Structure work assignments as clear, actionable items that the agent can complete independently following the defined workflow:
- Code implementation
- Test creation
- Documentation updates
- Git commits

## Expected Agent Behavior
With these instructions, agents should:
- Work through multiple tasks autonomously
- Make reasonable technical decisions
- Follow established code quality standards
- Create comprehensive tests and documentation
- Complete full development lifecycle for each task
- Continue iterating until all assigned work is done

## Customization
These files can be modified to:
- Add project-specific requirements
- Update technical standards as the project evolves
- Refine the autonomous development process
- Include new tools or frameworks
